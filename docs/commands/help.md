# NAME

`/help` – List all available slash commands with one-line descriptions.

# SYNOPSIS

```
/help
```

# DESCRIPTION

Displays a table of every registered slash command and its short summary.
Use `/man <command>` for detailed usage, options, and examples of a specific
command.

# EXAMPLES

- `/help` – Show the current command roster.
- `/man status` – Jump directly to the detailed documentation for `/status`.

# SEE ALSO

`/man`, `/status`, `/config`
